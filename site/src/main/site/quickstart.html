Pragmatach Quickstart

Firstly, clone the Pragmatach git repo, and build it. This will build and install all libraries.

			
mvn clean package install

To create the skeleton of a test project use the pragmatach maven archetype 


mvn archetype:generate                           \
  -DarchetypeGroupId=com.khubla.pragmatach       \
  -DarchetypeArtifactId=pragmatach-archetype     \
  -DarchetypeVersion=1.0-SNAPSHOT                \
  -DgroupId=com.khubla.exampleproject            \
  -DartifactId=myexampleproject

Alternatively, just run the supplied shell script:

sh create-example-project.sh

This will generate:

  • A maven pom.xml which includes the Pragmatach framework and some plugins
  • A simple controller and freemarker template for an index page
  • All other necessary resources to crete a simple Pragmatach project

To run the generated project, type:


cd myexampleproject
mvn clean package jetty:run

Then browse to:

http://localhost:8080
